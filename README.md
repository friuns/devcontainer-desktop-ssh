# Dev Container with Full Desktop and SSH

This project provides a development container with:
- **Full Desktop Environment**: Access via noVNC (port 6080) or VNC (port 5901)
- **SSH Access**: Connect via SSH on port 2222
- **Fluxbox Window Manager**: GUI applications will open on the desktop

## Features

- **Desktop Lite**: Fluxbox-based lightweight desktop
- **SSH Server**: Remote access via SSH
- **Universal Base**: Full Ubuntu with development tools
- **Common Utils**: Enhanced utilities and package management
- **Desktop Apps**: File manager, text editors, terminal, and more

## Pre-installed Applications

- **File Managers**: Thunar, Nautilus
- **Text Editors**: Mousepad, Gedit, Leafpad, Vim-gtk3
- **Terminals**: XFCE4 Terminal, XTerm
- **Web Browser**: Firefox
- **Development Tools**: Git, GitHub CLI, Zsh with Oh My Zsh

## Access Information

- **Desktop (Web)**: http://localhost:6080 (password: vscode)
- **VNC**: localhost:5901 (password: vscode)
- **SSH**: localhost:2222 (user: vscode)

## Development

This container is designed for developing GUI applications in a containerized environment with a full Linux desktop experience.
